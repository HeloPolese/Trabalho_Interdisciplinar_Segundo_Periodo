# Trabalho_Interdisciplinar_Segundo_Periodo
Este repositório contém o desenvolvimento do projeto interdisciplinar das disciplinas de Análise de Sistemas, Programação Orientada a Objetos I, UX Design e Banco de Dados I.

O sistema tem como objetivo gerenciar competições de corrida, inicialmente nas modalidades Maratonas e Trail Running, com possibilidade de expansão futura.
Permite que organizadores registrem eventos, gerenciem atletas inscritos, consultem competições e visualizem informações detalhadas de atletas e provas.
